Добрый день!
Вашему вниманию представлена примитивная программа фильтрации товара.
При старте программы все сгенерированные модели ноутбуков записываются в файл Laptops_Collection.txt
При Фильтрации отфильтрованные ноутбуки записываются в файл Found_Model.txt
После фильтрации при написании [Заново] будет использоваться изначально сгенерированная коллекция,
которую можно повторно фильтровать.
Объем HDD и RAM фильтруется по минимальному значению, то есть при указании числа 100 будет выведено всё, что больше него.
